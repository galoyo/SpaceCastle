Instructions.

LEFT / RIGHT arrow keys moves player.
DOWN arrow key: interact with objects such as characters, doors and dialogs.
i: Access the inventory.
z, x, c: Action key for an item. Confirm.
m: Main menu.
F12: Toggles full screen mode.
Holding SHIFT key then press + / - to increase / decrease the volume.

T KEY toggles cheat mode on / off.
H KEY increase health by 1.
L KEY increase air in lungs by 10.
